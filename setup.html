Warum warnt Windows?

Der Installer ist Freeware und hat kein teures EV-Code-Signing-Zertifikat. SmartScreen erkennt den Herausgeber daher nicht. Klicke auf „Weitere Informationen“ → „Trotzdem ausführen“. Der Build läuft transparent auf GitHub Actions mit offiziellem Python + Inno Setup — der Code ist einsehbar. Du kannst die Datei auch auf VirusTotal prüfen.

Nur Singleplayer: SweetCheat ist ausschließlich für Offline- & Solo-Spiele gedacht. Keine Multiplayer-Cheats, keine Anti-Cheat-Umgehung.
